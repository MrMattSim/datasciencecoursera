## SSH setup for automatic github authentication

####Create config file

```sh
$touch ~/.ssh/config
$vi ~/.ssh/config
```

```text
#MrMattSim account
Host github.com-MrMattSim
	HostName github.com
	User git #this must always be "git"
	IdentityFile ~/.ssh/id_rsa #or whatere was used for
```
